## ğŸ´ How to Fork WinkleGames

### 1. Fork the Repository

- Go to the WinkleGames GitHub page  
- Click the **Fork** button in the top-right corner  
- GitHub will create a copy under your account  

You now own your version.

---

### 2. Rename (Optional but Recommended)

To rebrand:

- Go to your forked repository  
- Click **Settings**  
- Change the repository name  
- Update the title inside `index.html`  

---

### 3. Customize It

You can now:

- Change colors in `style.css`  
- Edit layout in `index.html`  
- Add games inside the `/games` folder  
- Replace logos or icons in `/assets`  

Make it your own arcade.

---

### 4. Deploy with GitHub Pages (Optional)

If you want your version live:

- Go to **Settings â†’ Pages**  
- Select the `main` branch  
- Save  

GitHub will generate a public link for your site.
